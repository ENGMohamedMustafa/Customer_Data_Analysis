# Customer_Data_Analysis
This Jupyter notebook is designed for analyzing customer data to gain insights into sales performance, product categories, and customer behavior. It utilizes Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn for data manipulation and visualization.
Import Libraries: Essential libraries for data analysis are imported.
Read Dataset: The dataset is loaded from an Excel file.
Dataset Information: Basic information about the dataset is displayed, including data types and non-null counts.
Data Preprocessing: Steps include handling missing values, converting data types, and feature engineering.
Data Visualization: Various visualizations are created to illustrate sales by product category and other metrics.
Model Building: Predictive models may be built using machine learning algorithms from Scikit-learn.
Objectives
To understand sales trends across different product categories.
To preprocess and clean customer data for further analysis.
To visualize key metrics that drive business decisions.
Usage
Clone this repository and run the notebook in a Jupyter environment. Ensure all dependencies are installed, and update the dataset path as needed.

Dependencies
numpy
pandas
matplotlib
seaborn
scikit-learn
